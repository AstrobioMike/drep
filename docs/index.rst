dRep
====

de-replication of microbial genomes
-----------------------------------

The publication is `available on bioRxiv
<https://www.youtube.com/watch?v=dQw4w9WgXcQ>`_.

Source code is `available on GitHub
<https://github.com/MrOlm/drep>`_.

See links to the left for :doc:`installation` and :doc:`quick_start` instructions

Comments and suggestions can be sent to `Matt Olm <mattolm@berkeley.edu>`_
